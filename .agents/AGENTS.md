# Default Theme Gradient
The default gradient color for the fmp-client project is:
- **Tailwind Classes:** `bg-gradient-to-r from-[#A855F7] to-[#ff6118]`
- **CSS Inline Style:** `background: linear-gradient(to right, #A855F7, #ff6118)`

Always use this exact gradient when asked to apply a "gradient", "theme gradient color", or similar styling to buttons or UI elements, unless specified otherwise.
